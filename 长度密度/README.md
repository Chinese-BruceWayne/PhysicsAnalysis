# README

**数据txt格式**

## 文件名格式

* 四个表格按顺序分别命名为

	* table1.txt

	* table2.txt

	* table3.txt

	* table4.txt

## 文件内容格式

* 数据测量表格

|测量|测量次数1|测量次数2|测量次数3|....|
|:--:|:--:|:--:|:--:|:--:|
|变量1|11|12|13|...|
|变量2|21|22|23|...|
|变量3|31|32|33|...|
|...|...|...|...|...|

四个表格分别是4x4 4x4 1x6 2x3

* 文件结束给出仪器的分辨率（B类不确定度的根号三倍）

* **PS 密度测量表格中的两个数都是测量值与净重的差值**
